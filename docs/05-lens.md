# Lens 기능

Holonet의 Lens 기능을 사용하여 Kubernetes 클러스터를 시각화하고 관리할 수 있습니다.

## 🔭 Lens 뷰 열기

1. 상단 탭에서 **"🔭 Lens (K8s)"** 클릭
2. 클러스터 리소스가 표시됩니다

## 📦 네임스페이스

### 기본 설정

Lens 뷰는 **"dev"** 네임스페이스만 표시합니다.

### 네임스페이스 확인

상단에 **"dev"** 네임스페이스가 표시됩니다.

## 📊 리소스 타입

### Pods (파드)

실행 중인 컨테이너 인스턴스를 확인합니다.

**표시 정보:**
- **Name**: 파드 이름
- **Status**: 상태 (Running, Pending, Error 등)
- **Ready**: 준비된 컨테이너 수 (예: 2/2)
- **Restarts**: 재시작 횟수
- **Age**: 생성 후 경과 시간

**기능:**
- 파드를 클릭하면 상세 정보와 로그를 확인할 수 있습니다

### Deployments (디플로이먼트)

애플리케이션 배포 상태를 확인합니다.

**표시 정보:**
- **Name**: 디플로이먼트 이름
- **Status**: 상태
- **Ready**: 준비된 레플리카 수 (예: 3/3)
- **Available**: 사용 가능한 레플리카 수
- **Age**: 생성 후 경과 시간

### Services (서비스)

네트워크 서비스를 확인합니다.

**표시 정보:**
- **Name**: 서비스 이름
- **Status**: 상태
- **Type**: 서비스 타입 (ClusterIP, LoadBalancer 등)
- **Cluster IP**: 클러스터 내부 IP 주소
- **Age**: 생성 후 경과 시간

**기능:**
- 서비스를 클릭하면 상세 설정을 확인할 수 있습니다

## 🔍 리소스 상세 보기

### 상세 정보 확인

1. 리소스를 클릭합니다
2. 오른쪽 패널에 상세 정보가 JSON 형식으로 표시됩니다

### 파드 로그 보기

1. **Pods** 탭을 선택합니다
2. 파드를 클릭합니다
3. 오른쪽 패널 상단에 **"Logs"** 섹션이 표시됩니다
4. 최근 100줄의 로그가 표시됩니다

## 🔄 새로고침

**"🔄 Refresh"** 버튼을 클릭하여 리소스 목록을 업데이트합니다.

## 📋 리소스 정보

### 상태 색상

- 🟢 **초록색**: 정상 상태 (Running, Active)
- 🟡 **노란색**: 대기 중 (Pending)
- 🔴 **빨간색**: 오류 상태 (Error, Failed)

### 상세 정보 구조

리소스를 클릭하면 다음 정보가 표시됩니다:
- **metadata**: 이름, 네임스페이스, 라벨 등
- **spec**: 설정 정보
- **status**: 현재 상태

## 🎯 사용 예시

### 예시 1: 서비스 상태 확인

1. **Services** 탭 선택
2. 서비스 목록 확인
3. 특정 서비스를 클릭하여 상세 정보 확인

### 예시 2: 파드 로그 확인

1. **Pods** 탭 선택
2. 문제가 있는 파드 찾기 (상태가 Error인 파드)
3. 파드를 클릭하여 로그 확인

### 예시 3: 디플로이먼트 상태 모니터링

1. **Deployments** 탭 선택
2. Ready/Available 수 확인
3. 정상적으로 배포되었는지 확인

## ⚠️ 주의사항

1. **권한**: 클러스터 읽기 권한이 필요합니다
2. **네트워크**: 클러스터에 접근 가능해야 합니다
3. **성능**: 많은 리소스가 있으면 로딩에 시간이 걸릴 수 있습니다
